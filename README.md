# next-inspect

TypeScript npm package boilerplate.

## Scripts

- `npm run build` - Build to `dist/`
- `npm run dev` - Watch mode build
- `npm run typecheck` - Type-check project
- `npm run clean` - Remove `dist/`

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the package:
   ```bash
   npm run build
   ```
